Java集合总结

由于Java集合中有众多接口和实现类，本文只选取常用的集合。

Collection

Collection接口在java.util中，是所有单列集合的父接口，比如List和Set

常用方法：

  1. public boolean add(E e); 添加给定对象
  2. public void clear(); 清空集合
  3. public boolean remove(E e); 删除给定对象
  4. public boolean contains(E e); 判断集合中是否包含指定对象
  5. public boolean isEmpty(); 判断当前集合是否为空
  6. public int size(); 返回集合中元素的个数
  7. public Object[] toArray(); 将集合元素存储到数组中Collection集合在java.util中，是所有单列集合的父接口，比如List和Set

List

List集合接口中有四个常用的方法 1. add() 2. remove() 3. get() 4. set()

ArrayList

ArrayList是变长数组，类似于C++中的vector。注意<>中是泛型，只能是引用类型，而不能是基本类型

基本类型 包装类（引用类型） byte Byte short Short int Integer 需要记 long Long float Float double Double char Character 需要记 boolean Boolean

常用方法有：

  1. public boolean add(E e); 添加元素
  2. public E get(int index); 读取元素
  3. public E remove(int index); 删除元素
  4. public int size(); 获取集合中元素个数

LinkedList

LinkedList是List接口的另一个实现类

常用的方法： 一、与添加相关的方法

  1. public void addFirst(E e); 添加指定元素到链表的开头
  2. public void addLast(E e); 添加指定的元素到链表的结尾
  3. public void push(E e); 添加元素到此列表所表示的堆栈，就是添加元素到链表开头

二、与获取相关的方法

  1. public E getFirst(); 获取链表的第一个元素
  2. public E getLast(); 获取链表的最后一个元素

三、 删除相关的方法

  1. public E removeFirst(); 移除链表的第一个元素
  2. public E removeLast(); 移除链表的最后一个元素
  3. public E pop(); 列表所表示的堆栈中弹出一个元素， 就是弹出链表开头的元素

HashSet

HashSet是Set接口的实现类

Person类会默认继承Object类的hashCode方法，该方法返回对象的hash值

LinkedHashSet

LinkedHashSet是HashSet类的子类

和HashSet不同的是LinkedHashSet是有顺序的,按照存储的顺序

Map

Map<K,V>是与Collection完全不同的两个接口，key不允许重复，而value允许重复

常用方法：

  1. public V put(K key, V value); 把指定的键值对添加到集合中
  2. public V remove(Object key); 把指定的键对应的键值对从集合中删除，返回删除的元素的值
  3. public V get(Object key); 根据指定的键获取对应的值
  4. boolean containsKey(Object key); 判断集合中是否包含指定的键
  5. boolean Set<K> keySet(); 获取集合中的所有键，存储到Set集合中
  6. public Set<Map.Entry<K,V>> entrySet(); 获取map集合中的所有键值对，存储到Set集合中

常用类：String

特点： 1. 字符串是常量，在创建后不能更改 2. 字符串在效果上面相当于char[]字符数组，但是底层原理是byte[]字节数组 创建字符串的3+1种方式(三种构造方法一种直接创建) 1. public String(); 创建一个空白字符串 2. public String(char[] array); 根据字符数组，创建对应的字符串 3. public String(byte[] array); 根据字节数组，创建对应的字符串 6. String str = "hello"; 直接创建

常用方法有：

一、比较相关方法

  1. public boolean equals(Object obj); 比较两个字符串内容
  2. public boolean equalsIgnoreCase(String str); 忽略大小写，进行内容比较

二、与获取相关的方法

  1. public int length(); 获取字符串中含有的字符个数
  2. public String concat(String str); 将当前字符串和参数字符串进行拼接成为新的字符串
  3. public char charAt(int index); 获取指定索引位置的单个字符
  4. int indexOf(String str); 查找参数字符串在本字符当中首次出现的索引位置，没有返回-1

三、截取相关的方法

  1. public String substring(int index); 从索引位置一直到字符串结尾，返回新字符串
  2. public String substring(int begin, int end); 截取索引范围内的子串，[begin,end)，包含左边，不包含右边

四、转换相关的方法

  1. public char[] toCharArray(); 将当前字符串拆分为字符数组作为返回值
  2. public byte[] getBytes(); 获得当前字符串底层的字节数组
  3. public String replace(CharSequence oldString, CharSequence newString); 将所有oldString替换为newString

五、分割字符串的方法

  1. public String split(String regex); 按照参数的规则也就是正则表达式规则，将字符串切分为若干部分

常用类：String Builder

字符串缓冲区，可以看成是一个长度可以变化的字符串，底层是一个数组，但是和String不一样，没有被final修饰，初始时16个byte,超出容量就容量翻倍。

构造方法：

  1. StringBuilder(); 空参构造方法
  2. StringBuilder(String str); 初始化为指定字符串内容

成员方法：

  1. public StringBuilder append(); 加入任意类型数据
  2. public String toString(); 变为String类型字符串

StringBuilder可以和String相互转化：

  1. String --> StringBuilder: 使用StringBuilder的带参构造方法
  2. StringBuilder --> String: 使用StringBuilder的toString方法

工具类：Collections

Collections是集合工具类，用于操作集合

常用方法：

  1. public static <T> boolean addAll(Collection<T> c, T...elements) 往集合中添加一些元素
  2. public static void shuffle(List<?> list) 打乱顺序
  3. public static <T> void sort(List<T> list) 将集合中元素按照默认规则排序
  4. public static <T> void sort(List<T> list, Comparator<? super T>) 将集合中元素按照指定规则排序

工具类：Arrays

java.util.Arrays是一个与数组相关的工具类

public static String toString(数组); 将参数数组转化为字符串

public static void sort(数组); 按照默认升序排序

工具类：Math

java.lang.Math是数学相关的工具类

  1. public static double abs(double num); 获取绝对值

  2. public static double ceil(double num); 向上取整

  3. public static double floor(double num); 向下取整

  4. public static Long round(double num); 四舍五入

  5. Math.PI 代表圆周率

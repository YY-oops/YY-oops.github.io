Java代码在计算机中经历的三个阶段

获取Class对象的三种方式

  1. Class.forName("全类名")：将字节码文件加载进内存，返回Class对象

    多用于配置文件，将全类名定义在配置文件中，读取配置文件，加载类

  2. 类名.class: 通过类名的属性class获取

    多用于参数的传递

  3. 对象.getClass(): 通过对象中定义的方法获取其Class对象

    用于对已知对象获取字节码

重点：同一个字节码文件在一次程序运行过程中，只会被加载一次，不论什么方式获取的Class对象都是同一个

例子：Person类

接下来的反射知识以例子方式阐述，这里先放出Person类

获取对象的成员变量们

  1. Field[] getFields() 获取所有public修饰的成员变量

  2. Field getField(String name) 获取指定名称public修饰的成员变量

  3. Field[] getDeclaredFields() 获取所有的成员变量(需要通过暴力反射获取任意权限修饰符修饰的成员变量)

  4. Field getDeclaredField(String name) 获取指定名称的成员变量(需要通过暴力反射获取任意权限修饰符修饰的成员变量)

  5. Object get(Object object) 通过获取的成员变量获取成员变量的值

  6. set(Object obj, Object value) 通过获取的成员变量设置成员变量的值

获取对象的构造方法们

  1. Constructor<T> getConstructor(Class<?>... parameterTypes) 获取对象的有参构造方法

  2. Constructor<?>[] getConstructors() 获取对象的所有构造方法

  3. Constructor<?>[] getDeclaredConstructor()功能和获取成员变量的方法3类似，后面不再赘述

  4. Constructor<T> getDeclaredConstructor(Class<?>... parameterTypes)功能和获取成员变量的方法4类似，后面不再赘述

  5. newInstance() 使用获取的构造器方法创建对象

获取对象的成员方法们

  1. Method getMethod(String name, Class<?>... parameterTypes) 获取对象的指定名称的无参数方法

  2. Method[] getMethods() 获取所有public修饰的方法

  3. Method[] getDeclaredMethods()

  4. Method[] getDeclaredMethod(String name, Class<?>... parameterTypes)

  5. invoke(Object object) 通过获取的成员方法执行对象的方法
